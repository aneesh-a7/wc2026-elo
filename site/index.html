Why trust it — calibration

Accuracy is a trap: "always pick the favorite" already scores 56% on World Cup games, and the model scores the same. What matters is whether upsets happen as often as predicted. This reliability curve, from a leak-free backtest of the last five World Cups, is the real test.

Read it like this: each point is a group of matches where the model gave the favorite a certain win chance predicted. The vertical position is how often the favorite actually won observed. Points on the dashed line = perfectly calibrated. When the model says a favorite is only 35% likely, the underdog really does win about that often — the model takes the upset the right amount, instead of blindly backing the stronger name.

Elo computed from ~49,500 international matches, 1872–2026 (eloratings.net methodology: K-weighting by match importance, goal-difference multiplier, neutral-venue).
Backtest: frozen pre-tournament ratings, zero outcome leakage, no parameters tuned on test tournaments. Live-update variant reaches 59.7% three-way accuracy across 2006–2022.
Model calls on played games use each team's pre-tournament Elo — an honest grade, not hindsight. Data: openfootball · martj42/international_results · shootouts. Portfolio project, not affiliated with FIFA.
